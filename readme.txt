# GitHub User Search

A simple web application to search for GitHub users and display random profiles on the home page.

## Features

- Search for GitHub users by username.
- Display random GitHub user profiles on the home page.
- Responsive and minimal design.

## Technologies Used

- HTML
- CSS
- TypeScript
- GitHub API
 

## Usage

- Enter a GitHub username in the search input and click "Search" or press "Enter".
- The application will display the user's profile.
- Random GitHub user profiles are displayed on the home page when the application is loaded.
